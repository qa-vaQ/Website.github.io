This website is an attempt to alleviate my boredom while at school.
An ongoing project I have been working on for multiple years. This site allows me to pool games I have found so if I am bored at school I can play them.
Of course, since I am a generous person, I have also made it so others can also access these games.

The games themselves. In the dropdown menu above is a range of games from Pacman to Pong, and Mario to Tanks.
I should note that this website is undergoing a massive renovation. And as such, some of the pages are not up to the same standard as the rest of the website. While some of the pages may be a bit garish, all the games are still functiuonal on these outdated pages.
These games are not mobile-friendly by the way.
